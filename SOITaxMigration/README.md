# SOI Tax Stats - Migration Data
This script cleans the county to county migration data from https://www.irs.gov/statistics/soi-tax-stats-migration-data

# Output files
The output files have the following variables.
<ul>
  <li>fromstate</li>
  <li>fromcounty</li>
  <li>tostate</li>
  <li>tocounty</li>
  <li>statecode</li>
  <li>desc</li>
  <li>noreturns</li>
  <li>noexceptions</li>
  <li>agi</li>
  <li>file</li>
  <li>fromyear</li>
 </ul>
